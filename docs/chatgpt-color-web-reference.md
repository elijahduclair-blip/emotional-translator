# ChatGPT Color Web Reference

## Purpose

Use this file as a reference for interpreting language through the user's Theory of Alignment / Emotional Color Translator system.

Default assumption: the user's language may belong to an evolving color-climate vocabulary. Treat words as relational signals before treating them as isolated labels.

The system is not a diagnostic tool and not a claim that a word permanently means one color. It is a disciplined translator for relation, emotional climate, theme, and shade position.

## Core Idea

Meaning is relational.

A word, feeling, person, object, or theme becomes readable through context:

```text
source + filter -> theme -> color-climate route -> shade position
```

The system asks:

- What is the source?
- What filter is shaping it?
- What theme becomes visible?
- What color-climate does that route land in?
- Is the connection graph-supported, personally contextual, or unresolved?

## White Light, Filter, Theme

White light means raw possibility before interpretation.

A filter is the condition that makes the source readable. Filters can include mood, season, pressure, memory, relationship frame, role, history, science, art, religion, or personal context.

A theme is the visible relational pattern created by filtered light. A theme is not the whole person or object. It is the way something is shown under a filter.

Examples:

```text
white light + Halloween filter -> Halloween theme
white light + December filter -> December theme
person + friendship filter -> friendship-facing theme
same person + pressure filter -> pressure-facing theme
```

Boundary: themes are presentation climates, not permanent essence claims.

## Compass Rivers

Climate River:
Emotion moves like atmosphere, weather, rooms, crossroads, shadow, fog, pressure, warmth, and recovery.

Relationship River:
Connection is read through intimacy, tension, realism, masks, imperfection, attachment, and repair.

Identity / Pressure River:
Identity moves through pressure, reinvention, exposure, adaptation, and recovery. Multiple themes can coexist without contradiction.

## Theme Categories

Supported theme layers include:

- Religion: sacred and cultural meaning systems.
- Season: time, weather, cycle, and environmental mood.
- History: memory, institution, pressure, rupture, survival, and inheritance.
- Science: process, mechanism, evidence, structure, and revision.
- Arts: form, perception, symbol, performance, and composition.
- Foundation: experience, perception, emotion, adaptation, and behavior.
- Pattern: repetition, relation, context, similarity, difference, and network.
- Translation: signal, symbol, representation, color, shade, climate, meaning, and interpretation.
- Movement: change, response, pressure, growth, stability, regulation, adjustment, and transition.
- Identity: preference, tendency, habit, trait, character, personality, and individuality.

These are translator categories. They are not strict color synonyms.

## Shade Graph Axes

The shade graph uses three centered axes. The center is always `0`.

```text
X: cool -100 <- 0 -> warm +100
Y: dark -100 <- 0 -> light +100
Z: muted -100 <- 0 -> vivid +100
```

Interpretation:

- X shows color temperature or climate direction.
- Y shows light-depth, from hidden/dark/heavy to visible/light/open.
- Z shows saturation and clarity, from muted/foggy/ambiguous to vivid/clear/intense.

Examples:

```text
red -> warm positive X, slightly dark Y, vivid positive Z
gray -> near center Y, low/negative Z because it is muted
mist -> cool X, light Y, muted Z
ember -> warm X, dark-to-mid Y, vivid Z
```

## Why White And Black Can Feel Surprising

White and black can look surprising in the graph because the system uses actual RGB/HSL shade data, not only symbolic meaning.

White:
White often lands high on Y because it is light. But if the white is slightly blue, gray, or warm, it can move on X. Because white is low-saturation, it often lands negative on Z, meaning muted or low-color rather than vivid.

Black:
Black lands low on Y because it is dark. But a specific black can lean blue-black, brown-black, red-black, or gray-black, so it may move on X depending on its undertone. If a near-black color has a hue, the graph may show that undertone even when the human eye mainly reads it as black.

Rule:
For very low-light or low-saturation colors, treat X as undertone, not the whole meaning. The emotional meaning of black or white should still be checked through route, theme, and context.

## Color-Climate Anchor Families

Recurring anchors:

- Obsidian / shadow: reinvention, exposure, hiddenness, depth.
- Ember / amber: pressure, motion, release, heat, activation.
- Rose: intimacy, attachment, tenderness, connection.
- Midnight / ocean / blue: atmosphere, reflection, distance, calm, depth.
- Green / sage / pine: endurance, regulation, recovery, rootedness.
- Silver / mist / fog: revision, ambiguity, partial visibility, transition.

Use natural color language where possible. In this system:

```text
noun = source
adjective = condition or state
combination = relational meaning
```

Example:

```text
pine shadow
rose mist
ember pressure
silver threshold
```

## Emotion Shades

Emotion words can have graph-derived shades.

If an emotion routes to one color climate, use that route as its shade.

If an emotion routes to multiple color climates, mix the connected colors.

Examples:

```text
anger -> red + red-orange -> mixed anger shade
calm -> blue + green + blue-gray -> mixed calm shade
affection -> pink + pink-red -> mixed affection shade
```

Boundary:
Emotion shades are contextual routes, not strict synonyms and not diagnoses.

## Common And Neutral Word Shades

Common words can have graph-derived shades when they have supported color association routes.

Examples:

```text
coffee -> espresso + black + brown + charcoal -> mixed shade
apple -> green + red + yellow + green-yellow -> mixed shade
air -> blue + white + gray-white -> mixed shade
```

Neutral bridge words can have shades only when supported by a cited bridge route.

Examples:

```text
cash -> money -> green climate
ocean -> water -> blue climate
```

Unresolved words should not be forced.

Example:

```text
because -> unresolved
```

## Personal Profile Overlay

The shared graph is the foundation.

A personal profile is a local/private overlay that can shift interpretation for one person.

Personal context may include:

- anchors
- recurring pressures
- relationships or roles
- seasons or time periods
- memories or places
- boundaries or do-not-assume notes

Use this grammar:

```text
shared route + personal context -> personal climate shift
```

Boundary:
Personal context changes the read for that person only. It does not rewrite the shared graph and does not define permanent identity.

## Reasoning Rules For ChatGPT

When interpreting a word or phrase:

1. Check whether it is a direct color, emotion, common word, neutral bridge, theme, or personal-context word.
2. If it has graph-supported routes, explain those first.
3. If it has multiple supported routes, describe the blend as coexisting climates.
4. If it is a theme, explain the filter producing it.
5. If it is personal context, state that the read is an overlay.
6. If the route is not supported, call it unresolved instead of forcing meaning.

Use this output shape when helpful:

```text
Input:
Shared route:
Theme/filter:
Shade position:
Mixed components:
Emotional logic:
Boundary:
Unresolved checks:
```

## Falsifiability Checks

A reading is stronger when it can answer:

- What produced the route?
- What color or theme evidence supports it?
- What is the emotional logic?
- What boundary prevents overclaiming?
- What would make this reading weak or unresolved?

Mark a route weak or unresolved when:

- it only sounds poetic,
- it cannot name its filter,
- it has no graph-supported color/theme route,
- it overwrites the person or object,
- it treats a theme as permanent essence.

## Tone For Responses

Use language that is interpretive but disciplined.

Good:

```text
This reads as a mixed climate because the graph routes it through blue and green.
The route suggests regulation and reflection, not a fixed identity.
```

Avoid:

```text
This word objectively means blue.
This person is a blue person.
This proves the emotion.
```

The system should feel like a map, not a verdict.
